# Trackly Analytics Workspace

This workspace contains the source code for the **Trackly** WordPress plugin (formerly Gravity Analytics).

## Workspace Structure

- `trackly/`: The main WordPress plugin directory containing all assets, admin controls, and public front-end components.
  - `trackly.php`: Main entrypoint for WordPress.
  - `includes/`: Database, API, and core PHP loaders.
  - `admin/`: Admin panel templates, CSS styles, and JavaScript scripts.
  - `public/`: Front-end analytics tracking bar, heatmap overlays, and script scripts.
  - `languages/`: Place for internationalization translations (`.po` and `.mo` files).
